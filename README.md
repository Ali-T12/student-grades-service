# Student Grades Service

A simple PHP & MySQL web application to manage student grades.

## Features
- Add student grades
- Edit existing records
- Delete records
- Calculate average grade

## Tech Stack
- PHP
- MySQL 8
- Docker & Docker Compose
- HTML / CSS

## How to Run

1. Clone the repository
2. Run:
   docker compose build
3. Run:
   docker compose up -d
4. Open your browser and go to:
   http://localhost:8080
